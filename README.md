# GITFAT

Checking the size of the GitHub repositories in your organization

## Installation

```sh
brew install TechHoldingLLC/tap/gitfat
```
or 
```sh
brew tap TechHoldingLLC/tap
brew install gitfat
```

## Usage

Usage:
```sh
gitfat --org TechHoldingLLC --token ghp_mygithubtoken
```

And always:
```sh
gitfat -h
```
